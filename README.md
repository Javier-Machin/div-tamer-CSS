# div-tamer-CSS 

It's a simple CSS framework I made for TOP's CSS course.

Es un framework CSS bastante simple hecho como parte de un ejercicio en TOP.

The available classes are  / Las clases disponibles son:

* `navbar`
* `navbar-fixed`
* `footer`
* `footer-fixed`
* `nav-links`
* `right`
* `left`
* `grid-1-2-hor`
* `grid-1-2-ver`
* `grid-2-1-1`
* `grid-1-1-2`
* `grid-3`
* `grid-3x`
* `grid-4`
* `grid-4x`
* `main-container`
